# NumPy & Pandas Employee Analysis

This is a beginner-friendly data analysis project using **NumPy** and **Pandas**.

## Project Overview
The notebook focuses on:
- Data cleaning (missing values, outliers, skewed data)
- Exploratory data analysis using Pandas
- Correlation analysis and grouped insights

## Project Structure
- data/ → raw dataset
- notebooks/ → analysis notebook

## Tools Used
- Python
- NumPy
- Pandas
- Jupyter Notebook

## Notes
The dataset is highly right-skewed, so outlier handling techniques were applied carefully.
